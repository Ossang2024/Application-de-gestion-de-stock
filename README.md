# Application de Gestion de Stock

## Description

Cette application web permet de gérer le stockage des produits dans une entreprise.

Elle permet d'ajouter, modifier, supprimer et consulter les produits ainsi que de gérer les entrées et sorties de stock.

Le projet est développé avec le framework Laravel.

---

## Technologies utilisées

- PHP
- Laravel
- MySQL
- HTML
- CSS
- JavaScript

---

## Installation

1. Cloner le projet

git clone https://github.com/votre-nom/gestion-stock-laravel.git

2. Installer les dépendances

composer install

3. Configurer le fichier .env

4. Générer la clé Laravel

php artisan key:generate

5. Lancer le serveur

php artisan serve

---

## Fonctionnalités

- Gestion des produits
- Gestion des catégories
- Gestion des fournisseurs
- Entrées de stock
- Sorties de stock
- Historique des mouvements
